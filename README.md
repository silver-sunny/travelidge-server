## 🧾 프로젝트 소개

자체운영과 네이버 커머스를 같이 연동해서 사용할수있는 온라인 예약 관리 시스템입니다.
사용자에게는 간편한 
관리자에게는 편하게 같이 작성ㅎ

음료 한잔의 소소한 행복을 더욱 편리하게 해주는 온라인 카페 주문 시스템입니다.  
사용자에게는 간편한 메뉴확인과 빠른 주문이라는 편의성, 포인트 적립이라는 이득을 제공합니다.  
관리자에게는 시스템 사용의 이득을 통한 사용자 모집 효과, 간편한 주문관리를 제공합니다.

- `SpringBoot`와 `Spring Data JPA`를 사용해 기본적인 REST API를 구현하였습니다
- 네이버 커머스와의 주문 싱크를 맞추기 위해 spring batch를 사용하였습니다.

- `Docker`, `Oracle Cloud`, `Jenkins` 등을 이용해 서버를 배포했습니다.

##  주요 기능

###  공통
- 로그인
- 상품 검색, 목록 및 상세 조회


### 사용자

- 회원가입
- 장바구니 기능
- 주문 취소, 반품 처리
- 리뷰 작성 및 관리
